XSLT/XPath Reference: XSLT elements, EXSLT functions, XPath functions, XPath axes

The EXSLT Math package provides functions for working with numeric values and comparing nodes. The namespace for the Math package is http://exslt.org/math.

math:highest()

math:highest() returns the node in the specified node-set with the highest value (where the highest value calculated using math:max()).

math:lowest()

math:lowest() returns the node in the specified node-set with the lowest value (where the lowest value calculated using math:min()).

math:max()

math:max() returns the maximum value of a node-set.

math:min()

math:min() returns the minimum value of a node-set.
